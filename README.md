# WebDalmuti

## Description [프로젝트 소개]

React.js, Express 서버를 활용하여 달무티 보드게임을 웹으로 구현하고
다수의 유저가 실시간으로 게임을 플레이할 수 있는 프로젝트입니다.

## Dependencies

-   typescript
-   React, styled-component, Recoil
-   Express, axios, dotenv, mongoose
